# 📅 Day 1 – Python Basics

## 🎯 Objective

To understand the fundamentals of Python programming and basic concepts required for Data Science.

---

## 📘 Topics Covered

### 1. Variables

* Declaring variables
* Assigning values
* Swapping variables
* Reassigning values
* Understanding errors (NameError)

### 2. Data Types

* Integer, Float, String, Boolean
* Type checking using type()
* Checking instance using isinstance()
* Understanding NoneType

### 3. Type Conversion

* Converting between string, integer, float, and boolean
* Handling invalid conversions (ValueError)
* Taking user input and processing it

### 4. String Manipulation

* Case conversion (upper, lower)
* String slicing
* Reversing strings
* Searching and replacing
* Splitting and trimming strings

### 5. String Formatting

* f-strings
* format() method
* Decimal formatting
* Padding and alignment

### 6. Python Lists

* Creating lists
* Accessing elements
* Adding and removing elements
* Sorting lists
* List slicing
* Copying and merging lists

---
