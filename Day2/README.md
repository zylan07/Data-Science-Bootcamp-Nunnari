# 📅 Day 2 – Python & NumPy Practice

## 🎯 Objective

To strengthen Python fundamentals and understand the basics of NumPy for data science applications.

---

## 📘 Topics Covered

### 1. Python Data Structures

* Lists, Tuples, Dictionaries, Sets
* Operations like append, insert, remove, union, intersection

### 2. Control Flow

* Conditional statements (if, elif, else)
* Loops (for, while, break, continue)

### 3. Functions

* Function definition and usage
* Arguments and return values
* Default parameters

### 4. Introduction to NumPy

* 1D and 2D arrays
* Array attributes (shape, size, dtype)
* Data types
* Indexing and slicing
* Mathematical operations

---

## 💻 Implementation

All concepts were implemented using Python in Google Colab by recreating instructor demonstrations and practicing each topic step-by-step.
